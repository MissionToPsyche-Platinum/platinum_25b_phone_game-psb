# 25B Phone-Based Psyche Game (2026)

## Project Description
The objective of this project is to improve (NASA Web-Based Game 15A 2020) Survive to Psyche's accessibility, engagement, and performance for mobile users by redesigning its interface, optimizing its assets, and ensuring seamless browser-based functionality.
